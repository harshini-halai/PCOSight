# PCOSight – Smart Insights for Women’s Health

PCOSight is a healthcare analytics web application that predicts PCOS risk based on user health inputs and provides BMI analysis, awareness resources, and dashboard analytics.

## Features
- PCOS risk prediction
- BMI calculation
- Dashboard analytics
- Pie chart visualization
- SQLite database integration
- Responsive healthcare UI
- Awareness and health recommendations

## Technologies Used
- HTML
- CSS
- JavaScript
- Flask
- SQLite
- Chart.js

## Project Modules
- Home Page
- Prediction System
- Result Analytics
- Dashboard
- Awareness Page
